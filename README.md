# SimpleOS-A-Minimal-OS-Booting-with-GRUB
SimpleOS is a minimal OS built with Assembly and C++, booting with GRUB to display a message. It includes a basic bootloader (`boot.asm`), a simple kernel (`kernel.cpp`), a linker script (`linker.ld`), and a loader.
